Introduction
This platform performs automated maritime surveillance on SAR images.
 It detects ships and oil spills using deep learning models and visualizes results on an interactive DeepZoom viewer. The goal is to reduce manual image scanning and support faster decision-making.
WORKFLOW
The system follows a client-server architecture where users interact through a React frontend with Firebase authentication. Uploaded SAR images are sent to the FastAPI backend, where they are tiled into Deep Zoom Images (DZI) and processed through two ML models — Deformable DETR for ship detection and TransUNet for oil-spill segmentation. The resulting detections are stored in MongoDB and retrieved back to the React viewer, where OpenSeadragon overlays the bounding boxes/masks on the high-resolution image.

DEFORMABLE DETR
Deformable DETR uses ResNet-50 as its backbone to extract feature maps from the SAR image.
 
 
 Finally, the model outputs accurate bounding boxes and class predictions for detected ships.
The encoder applies deformable self-attention, focusing only on a few key sampling points instead of the whole image. The decoder uses deformable cross-attention to refine object queries against selected feature locations.

TRANSUNET
TransUNet combines U-Net (CNN) with a Transformer encoder for medical/segmentation tasks.
The CNN encoder extracts low-level spatial features, and the Transformer captures long-range global context. These features are passed into a U-Net decoder that performs upsampling with skip connections.
The final output is a precise segmentation mask, making it ideal for pixel-accurate oil spill detection.

CONCLUSION
The Deep SAR Monitor System successfully automates ship detection and oil spill segmentation from high-resolution SAR satellite imagery. By integrating Deformable DETR for object detection and TransUNet for segmentation, the platform achieves accurate identification of maritime activities and environmental hazards. With FastAPI backend, MongoDB storage, and a DeepZoom-based React interface, users can visualize and inspect gigapixel satellite images in real time.
 This system reduces manual analysis effort, improves decision-making speed, and demonstrates how AI can enhance maritime safety, environmental monitoring, and remote sensing workflows.
